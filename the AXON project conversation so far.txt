What Is It?
AXON is a PowerShell-based AI agent tool. It runs entirely in your terminal with no external dependencies. The concept is simple but powerful: the AI is the brain, PowerShell is the hands. You talk to the AI, the AI decides what to do, and AXON carries it out on your machine — with your permission.

How It Works (The Core Loop)
You type something → AXON sends it to your chosen AI with full context → AI responds, sometimes with embedded commands using special tags → AXON parses those tags, checks safety, asks your approval if needed → executes → logs everything.

The Safety System (7 Layers)

Blocklist — hardcoded protected paths (System32 etc.) + your own custom ones. Fully resolved before checking so no path tricks bypass it.
Dangerous Cmdlet Scanner — catches things like Remove-Item -Recurse, registry edits, self-modification attempts before they run.
Dry Run — simulates the code first, shows you plain-English summary of what it will do.
Confirmation Tiers — 🟢 auto-approve / 🟡 ask user / 🔴 hard block, silently rejected.
Rate Limiter — max 5 code executions per minute so the AI can't spam actions.
Emergency Brake — /brake command or hotkey halts everything instantly.
Immutable Script — AXON cannot modify itself. Ever. The safety section is wrapped in #SAFETY_START / #SAFETY_END markers and is always stripped from the copy sent to the AI — so the AI never even sees the cage walls.
